# Sistema de Inventario Florícola

Proyecto desarrollado con React + Node.js para control de inventario floral.

## Funciones

* Registro de productos
* Control de stock
* Precio alto y bajo
* Fecha automática
* Dashboard visual

## Tecnologías

* React
* Node.js
* Express
* Git

  ## Instalación

npm install
npm start

backend = node app.js
frontend = npm start
